# ember-electron

https://github.com/felixrieseberg/ember-electron

Very simple way to create electron app for desktop with emberjs

## Create ember app

1. Install ember-cli, yarn add ember-cli -g
2. ember new 
3. ember install ember-electron
4. ember electron

Wait for the magic.
